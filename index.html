L-01 Inversare ArrayList fără Collections.reverse() Easy

Cerință

Scrie <T> void reverseList(List<T> list) care inversează in-place, cu get(), set(), size().

[1, 2, 3, 4, 5]  →  [5, 4, 3, 2, 1]
["a", "b", "c"]  →  ["c", "b", "a"]
Doi pointeri: swap i cu size-1-i, merge până la jumătate.
public static <T> void reverseList(List<T> list) {
    int n = list.size();
    for (int i = 0; i < n / 2; i++) {
        T temp = list.get(i);
        list.set(i, list.get(n - 1 - i));
        list.set(n - 1 - i, temp);
    }
}
// O(n) timp, O(1) spațiu

L-03 Găsește Min și Max fără Collections Easy

Cerință

Returnează int[]{min, max} dintr-o listă. Excepție dacă e goală. Fără Collections.min/max sau Streams.

Input:  [3, 7, 1, 9, -2, 8]
Output: [-2, 9]
Inițializează min/max cu primul element, parcurge restul comparând.
public static int[] findMinMax(List<Integer> list) {
    if (list.isEmpty()) throw new IllegalArgumentException();
    int min = list.get(0), max = list.get(0);
    for (int i = 1; i < list.size(); i++) {
        int v = list.get(i);
        if (v < min) min = v;
        if (v > max) max = v;
    }
    return new int[]{min, max};
}
L-04 Verifică dacă lista e palindrom Easy

Cerință

Scrie <T> boolean isPalindrome(List<T> list).

[1, 2, 3, 2, 1] → true      [1, 2, 3] → false
["a","b","a"]  → true      []        → true
Doi pointeri: i de la 0, j de la size-1. Compară cu .equals() nu ==.
public static <T> boolean isPalindrome(List<T> list) {
    int i = 0, j = list.size() - 1;
    while (i < j) {
        if (!list.get(i).equals(list.get(j))) return false;
        i++; j--;
    }
    return true;
}
// .equals() nu == (pentru Integer cache > 127 e capcană!)
L-05 Merge două liste sortate Easy

Cerință

Date două liste sortate crescător, returnează o nouă listă sortată. Fără sort().

a=[1, 3, 5, 7]  b=[2, 4, 6, 8, 10]
→ [1, 2, 3, 4, 5, 6, 7, 8, 10]
Doi indecși. La fiecare pas adaugi cel mai mic, avansezi indexul. La final adaugi restul.
public static List<Integer> mergeSorted(
        List<Integer> a, List<Integer> b) {
    List<Integer> res = new ArrayList<>(a.size() + b.size());
    int i = 0, j = 0;
    while (i < a.size() && j < b.size()) {
        if (a.get(i) <= b.get(j)) res.add(a.get(i++));
        else res.add(b.get(j++));
    }
    while (i < a.size()) res.add(a.get(i++));
    while (j < b.size()) res.add(b.get(j++));
    return res;
}
// O(n + m) timp

[1, 2, 3, 4, 5], k=2  →  [3, 4, 5, 1, 2]
[10, 20, 30], k=5      →  [30, 10, 20]  // 5%3=2
Normalizează k%size. Trick: 3 reverse-uri — reverse(0,k), reverse(k,n), reverse(0,n).
// Varianta elegantă: 3 reverse-uri — O(n), O(1) spațiu
public static <T> void rotateLeft(List<T> list, int k) {
    if (list.isEmpty()) return;
    k = k % list.size();
    Collections.reverse(list.subList(0, k));
    Collections.reverse(list.subList(k, list.size()));
    Collections.reverse(list);
}
// [1,2|3,4,5] → [2,1|5,4,3] → [3,4,5,1,2] ✓

L-08 Partition List cu Predicate Medium

Cerință

Primește List<T> + Predicate<T>, returnează [matching, nonMatching].

[1, 2, 3, 4, 5, 6], n -> n%2==0  →  [[2,4,6], [1,3,5]]
Două ArrayList-uri, iterează, adaugă după predicate.test(item).
public static <T> List<List<T>> partition(
        List<T> list, Predicate<T> pred) {
    List<T> yes = new ArrayList<>(), no = new ArrayList<>();
    for (T item : list)
        (pred.test(item) ? yes : no).add(item);
    return List.of(yes, no);
}
L-09 Merge Sort pe LinkedList Hard

Cerință

Merge Sort pe LinkedList<Integer> fără conversie la array. O(n log n). Fără Collections.sort().

[38, 27, 43, 3, 9, 82, 10]  →  [3, 9, 10, 27, 38, 43, 82]
Split cu subList, copiază în LinkedList-uri noi, recursiv, merge cu doi iteratori.
public static LinkedList<Integer> mergeSort(LinkedList<Integer> list) {
    if (list.size() <= 1) return list;
    int mid = list.size() / 2;
    var left  = mergeSort(new LinkedList<>(list.subList(0, mid)));
    var right = mergeSort(new LinkedList<>(list.subList(mid, list.size())));
    return merge(left, right);
}

private static LinkedList<Integer> merge(
        LinkedList<Integer> a, LinkedList<Integer> b) {
    LinkedList<Integer> res = new LinkedList<>();
    var itA = a.iterator(); var itB = b.iterator();
    Integer vA = itA.hasNext() ? itA.next() : null;
    Integer vB = itB.hasNext() ? itB.next() : null;
    while (vA != null && vB != null) {
        if (vA <= vB) { res.add(vA); vA = itA.hasNext()?itA.next():null; }
        else          { res.add(vB); vB = itB.hasNext()?itB.next():null; }
    }
    if (vA!=null) res.add(vA); if (vB!=null) res.add(vB);
    itA.forEachRemaining(res::add); itB.forEachRemaining(res::add);
    return res;
}

S-03 Diferența simetrică (XOR pe mulțimi) Easy

Cerință

Scrie symmetricDifference(Set<T> a, Set<T> b) care returnează elementele prezente în A sau B, dar nu în ambele (echivalentul XOR pe mulțimi).

A = {1, 2, 3, 4}    B = {3, 4, 5, 6}
symmetricDiff → {1, 2, 5, 6}
Union minus Intersection. Sau: (A \ B) ∪ (B \ A). Poți refolosi metodele din S-01!
public static <T> Set<T> symmetricDiff(Set<T> a, Set<T> b) {
    Set<T> result = new HashSet<>(a);
    for (T elem : b) {
        if (!result.add(elem)) {  // add returns false if existed
            result.remove(elem);   // was in both → remove
        }
    }
    return result;
}

// Alternativ cu operații din S-01:
// union(difference(a, b), difference(b, a))

S-05 Primul caracter ne-repetat — LinkedHashMap Easy

public static Character firstUnique(String s) {
    Map<Character, Integer> freq = new LinkedHashMap<>();
    for (char c : s.toCharArray())
        freq.merge(c, 1, Integer::sum);

S-07 TreeSet — Interval Scheduling (Calendar) Hard

Cerință

record Interval(int start, int end) {}

public class Calendar {
    private TreeSet<Interval> bk =
        new TreeSet<>(Comparator.comparingInt(Interval::start));

    public boolean book(Interval i) {
        Interval lo = bk.floor(i), hi = bk.ceiling(i);
        if (lo != null && lo.end() > i.start()) return false;
        if (hi != null && hi.start() < i.end()) return false;
        bk.add(i); return true;
    }
}

Q-01 Stack cu Deque — Verificare Paranteze Easy

Cerință

Verifică paranteze echilibrate ()[]{} cu ArrayDeque (nu Stack legacy).

"({[]})"→true   "([)]"→false   ""→true   "((("→false
Push closing bracket corespunzător. La closing, verifică top == ea. Final: stack gol = valid.
public static boolean isBalanced(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c=='(') stack.push(')');
        else if (c=='[') stack.push(']');
        else if (c=='{') stack.push('}');
        else if (c==')'||c==']'||c=='}') {
            if (stack.isEmpty()||stack.pop()!=c) return false;
        }
    }
    return stack.isEmpty();
}
Q-02 Inversează o Queue folosind un Stack Easy

Cerință

Dat un Queue<Integer>, inversează ordinea elementelor folosind un Deque ca stack auxiliar. Modifică queue-ul in-place.

Input:  Queue: [1, 2, 3, 4, 5]  (front=1)
Output: Queue: [5, 4, 3, 2, 1]  (front=5)
Golește queue-ul în stack (poll → push). Apoi golește stack-ul înapoi în queue (pop → offer). LIFO inversează FIFO.
public static void reverseQueue(Queue<Integer> queue) {
    Deque<Integer> stack = new ArrayDeque<>();

    // Queue → Stack (FIFO → LIFO = inversare)
    while (!queue.isEmpty())
        stack.push(queue.poll());

    // Stack → Queue (elementele ies invers)
    while (!stack.isEmpty())
        queue.offer(stack.pop());
}
// O(n) timp, O(n) spațiu
Q-03 Generează numerele binare de la 1 la N cu Queue Easy

Cerință

public static List<String> generateBinary(int n) {

        result.add(front);
        queue.offer(front + "0");
        queue.offer(front + "1");
    }
    return result;
}

Q-04 Hot Potato — Simulare cu Queue (Josephus) Easy

Cerință

N persoane stau în cerc. Se numără până la k, persoana la care se oprește iese. Se repetă până rămâne una singură. Returnează câștigătorul. Simulează cu Queue.

Input:  ["Ana","Bob","Carol","Dan","Eva"], k=3
Runde:  elimină Carol → Dan → Bob → Eva
Output: "Ana"
Pune toți în Queue. Repetă: mută k-1 persoane de la front la back (poll→offer), apoi poll pe a k-a (eliminată). Repetă până size == 1.
public static String hotPotato(List<String> names, int k) {
    Queue<String> circle = new LinkedList<>(names);

    while (circle.size() > 1) {
        // Pasează "cartofului fierbinte" de k-1 ori
        for (int i = 0; i < k - 1; i++)
            circle.offer(circle.poll()); // front → back
        circle.poll(); // al k-lea iese din joc
    }
    return circle.poll();
}
// Clasica problemă Josephus simulată cu Queue circular
Q-05 PriorityQueue — Top K Elemente Medium

Cerință

Cele mai mari k elemente din array, cu PriorityQueue (min-heap de size k). O(n log k).

[3, 1, 5, 12, 2, 11, 7], k=3  →  [7, 11, 12]
Min-heap de size k. Dacă size < k → add. Dacă elem > peek() → poll + add.
public static List<Integer> topK(int[] arr, int k) {
    PriorityQueue<Integer> heap = new PriorityQueue<>(k);
    for (int n : arr) {
        if (heap.size() < k) heap.offer(n);
        else if (n > heap.peek()) { heap.poll(); heap.offer(n); }
    }
    List<Integer> res = new ArrayList<>(heap);
    Collections.sort(res);
    return res;
}
Q-06 Sliding Window Maximum — Deque O(n) Hard

Cerință

Array + fereastră k → maximul din fiecare fereastră. O(n).

[1, 3, -1, -3, 5, 3, 6, 7], k=3  →  [3, 3, 5, 5, 6, 7]
ArrayDeque cu indecși, descrescător. Elimină din spate valorile ≤ noul elem. Elimină din față cei ieșiți din fereastră.
public static int[] maxSlidingWindow(int[] nums, int k) {
    int[] res = new int[nums.length - k + 1];
    Deque<Integer> dq = new ArrayDeque<>();
    for (int i = 0; i < nums.length; i++) {
        while (!dq.isEmpty() && dq.peekFirst() < i-k+1) dq.pollFirst();
        while (!dq.isEmpty() && nums[dq.peekLast()] <= nums[i]) dq.pollLast();
        dq.offerLast(i);
        if (i >= k-1) res[i-k+1] = nums[dq.peekFirst()];
    }
    return res;
}
// Fiecare element intră/iese maxim o dată → O(n)
Q-07 Task Scheduler — PriorityQueue + Cooldown Hard

Cerință

Task-uri cu cooldown n între execuțiile aceluiași task. Returnează timpul minim.

tasks=['A','A','A','B','B','B'], n=2  →  8
// A → B → idle → A → B → idle → A → B
Greedy: max-heap pe frecvențe + Queue pt cooldown tracking.
public static int leastInterval(char[] tasks, int n) {
    int[] freq = new int[26];
    for (char t : tasks) freq[t-'A']++;
    PriorityQueue<Integer> maxH =
        new PriorityQueue<>(Collections.reverseOrder());
    for (int f : freq) if (f>0) maxH.offer(f);
    Queue<int[]> cd = new LinkedList<>();
    int time = 0;
    while (!maxH.isEmpty() || !cd.isEmpty()) {
        time++;
        if (!maxH.isEmpty()) {
            int rem = maxH.poll()-1;
            if (rem>0) cd.offer(new int[]{rem, time+n});
        }
        if (!cd.isEmpty() && cd.peek()[1]==time)
            maxH.offer(cd.poll()[0]);
    }
    return time;
}

Q-09 Simulează o coadă de printare Easy

Simulează o coadă FIFO de printare: adaugă documente cu offer(), procesează cu poll(), returnează ordinea procesării.

Adaugă: "doc1", "doc2", "doc3"
Procesează 2: → ["doc1", "doc2"]

Procesează rest: → ["doc3", "doc4"]
LinkedList sau ArrayDeque — ambele implementează Queue.
public static List<String> processPrintQueue(String[] docs) {
    Queue<String> printer = new ArrayDeque<>();
    List<String> processed = new ArrayList<>();
    for (String doc : docs)
        printer.offer(doc);
    while (!printer.isEmpty())
        processed.add(printer.poll());
    return processed;
}
// FIFO: primul adăugat = primul procesat
Q-10 Interclasează două Queue-uri Easy

Cerință

Primești două Queue-uri. Returnează un nou Queue care alternează elementele: unul din Q1, unul din Q2. Restul se adaugă la final.

Q1: [1, 3, 5]    Q2: [2, 4, 6, 8]
Output: [1, 2, 3, 4, 5, 6, 8]
Cât timp ambele au elemente, poll alternativ. Apoi golește pe cel rămas.
public static Queue<Integer> interleave(
        Queue<Integer> q1, Queue<Integer> q2) {
    Queue<Integer> res = new ArrayDeque<>();
    while (!q1.isEmpty() && !q2.isEmpty()) {
        res.offer(q1.poll());
        res.offer(q2.poll());
    }
    while (!q1.isEmpty()) res.offer(q1.poll());
    while (!q2.isEmpty()) res.offer(q2.poll());
    return res;
}

public static boolean areAnagrams(String a, String b) {

    for (char c : a.toCharArray())

X-01 Two Sum — HashMap lookup O(n) Easy

Cerință

Doi indecși care însumează target. O(n).

nums=[2, 7, 11, 15], target=9  →  [0, 1]  // 2+7=9

public static int[] twoSum(int[] nums, int target) {
    Map<Integer,Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int comp = target - nums[i];
        if (map.containsKey(comp))
            return new int[]{map.get(comp), i};
        map.put(nums[i], i);
    }
    throw new IllegalArgumentException("No solution");
}

X-06 FlattenIterator — Iterator custom lazy Hard

Cerință

Implementează Iterator<T> care primește List<List<T>> și iterează flat, lazy (fără pre-flatten).

// next() → 1, 2, 3, 4, 5, 6 din [[1,2],[],[3],[4,5,6]]
Doi iteratori: outer (pe liste) + inner (pe elemente). hasNext() avansează la sublista non-vidă.
public class FlattenIterator<T> implements Iterator<T> {
    private final Iterator<List<T>> outer;
    private Iterator<T> inner;

    public FlattenIterator(List<List<T>> lists) {
        outer = lists.iterator();
        inner = Collections.emptyIterator();
    }
    @Override
    public boolean hasNext() {
        while (!inner.hasNext() && outer.hasNext())
            inner = outer.next().iterator();
        return inner.hasNext();
    }
    @Override
    public T next() {
        if (!hasNext()) throw new NoSuchElementException();
        return inner.next();
    }
}

X-10 Unește două liste sortate Easy

Cerință

Primești două liste sortate crescător. Returnează o nouă listă sortată care conține toate elementele.

A: [1, 3, 5, 7]    B: [2, 4, 6, 8]
Output: [1, 2, 3, 4, 5, 6, 7, 8]
Doi pointeri: compară elementele curente, adaugă pe cel mai mic, avansează pointerul.
public static List<Integer> mergeSorted(
        List<Integer> a, List<Integer> b) {
    List<Integer> res = new ArrayList<>();
    int i = 0, j = 0;
    while (i < a.size() && j < b.size()) {
        if (a.get(i) <= b.get(j)) res.add(a.get(i++));
        else res.add(b.get(j++));
    }
    while (i < a.size()) res.add(a.get(i++));
    while (j < b.size()) res.add(b.get(j++));
    return res;
}
// Merge sort step — O(n+m) timp, O(n+m) spațiu
